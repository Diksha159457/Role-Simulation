🌙 Daily Reflection Tree

A deterministic, LLM-free end-of-day reflection tool based on psychological research.

🧠 Evidence-Based 🚀 Production-Ready 🐳 Docker Support ⚡ Zero Dependencies

About This Project

This tool walks employees through three evidence-based psychological axes to help them reflect on their day.

🎯 Axis 1: Locus of Control

Measures whether you attribute outcomes to internal agency or external forces (Rotter, 1954)

💡 Axis 2: Orientation

Distinguishes between entitlement and contribution mindsets (Campbell et al., 2004)

🌍 Axis 3: Radius

Maps perspective from self-centric to transcendent (Maslow, 1969)

Try It Yourself

Run the CLI tool locally to experience the full reflection journey

python agent/agent.py

Key Features

🎯

Deterministic Design

No LLM calls at runtime. Every reflection is predictable, trustworthy, and consistent.

📊

40-Node Decision Tree

Comprehensive branching logic with 11 questions, 8 reflections, and personalized summaries.

🔧

Production-Ready

REST API, Docker support, and deployment guides for Heroku, Railway, and Render.

📚

Evidence-Based

Built on research from Rotter (1954), Dweck (2006), Maslow (1969), and others.

⚡

Zero Dependencies

Core logic runs on pure Python 3.8+ with no external libraries required.

🧪

Comprehensive Testing

100% test coverage with automated persona validation and signal verification.

API Documentation

🔐 Authentication Required

All API endpoints require an API key for authentication.

Default API Key: demo-key-12345

Set custom key via environment variable: API_KEY=your-secret-key

GET /tree

Retrieve the full JSON tree structure (40+ nodes)

GET /stats

Get tree statistics and metadata

Example: Using API Key in Header

curl http://localhost:5000/tree \
  -H "X-API-Key: demo-key-12345"

Example: Using API Key in Query Parameter

curl "http://localhost:5000/stats?api_key=demo-key-12345"

Example: Unauthorized Request (No API Key)

curl http://localhost:5000/tree
# Returns: {"error": "API key required", "message": "..."}
