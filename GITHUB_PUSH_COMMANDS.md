## ❓ Troubleshooting

**Q: GitHub still not showing all files?**
A: Run `git status` to check. If files are listed as "untracked", run `git add .` again.

**Q: Push rejected?**
A: Run `git pull origin main --rebase` then `git push origin main`

**Q: Web page shows 404?**
A: Make sure `templates/` folder exists and contains `index.html`
